# Churchill Crisis Leadership

基于温斯顿·丘吉尔战时领导力的系统化技能包，将 Churchill 的危机语言、战略节奏、团队构建、决策引擎算法化，适用于企业危机、战略转型、组织变革等场景。

## 🚀 快速开始

```bash
# 获取技能
openclaw skill install churchill-crisis-leadership

# 使用危机讲话生成
openclaw skill run churchill-crisis-leadership speech --crisis "产品市场下滑" --audience "员工"

# 启动决策节奏
openclaw skill run churchill-crisis-leadership decision-start --question "是否裁员？" --deadline 72
```

## 📦 核心模块

1. **Crisis Speech Engine** - 生成 Churchill 式危机动员讲话
2. **Strategic Rhythm Controller** - 维持长期危机中的战略节奏
3. **Crisis Team Architect** - 快速组建高效危机响应团队
4. **Decision Rhythm Engine** - 平衡速度与质量的决策流程
