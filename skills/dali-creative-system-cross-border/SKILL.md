## 📐 核心算法与公式

### 1. 偏执狂批判法创意引擎

**原理**：利用"controlled paranoia"让大脑跳出常规联想，发现隐藏的双重/多重意象。

**5步流程**：

1. **问题定义**：明确创意目标（如"新 logo 需要体现速度与安全"）
2. **诱导偏执**（10-15分钟）：
   - 旋转椅子 30 圈
   - 或长时间凝视一个相关图像（如汽车）
   - 或进入半催眠状态（闭眼，听单调声音）
3. **双重意象捕捉**：列出至少 10 个可能的双重解释（速度+安全 → 飞机+鸟巢？盾牌+翅膀？）
4. **精确投射**：选择 top 2-3，用精细技法（写实绘画/3D建模）实现
5. **测试与迭代**：用户测试，选择 recall_score 最高的

**量化公式**：
```
creative_score = (delusion_intensity × 0.4) + (technical_precision × 0.3) + (symbol_density × 0.3)
```
- delusion_intensity: 0-1，意象的意外程度（越出人意表越高）
- technical_precision: 0-1，执行完成度（达利油画 0.95）
- symbol_density: 0-1，每单位作品中的隐藏符号数量

**达利案例**《记忆的永恒》：
- delusion_intensity: 0.9（软表 = 时间融化，颠覆常识）
- technical_precision: 0.95（写实油画技法）
- symbol_density: 0.85（蚂蚁、橄榄树、烂脸、软表、坚硬的钟）
- **score ≈ 0.92** → 顶级

**应用示例**：设计"速度+安全" logo
- delusion_intensity: 0.7（鸟巢+飞机？一般）
- precision: 0.9（精细矢量）
- symbol_density: 0.6（2-3个隐含符号）
- **score ≈ 0.75** → 良好

### 2. 潜意识挖掘系统

**达利的潜意识工作流**：
1. **梦境记录**：床头放纸笔，醒来 5 分钟内记录（语音辅助）
2. **自由联想**：对每个元素展开联想，不评判
   - 例如：梦境有"钥匙" → 联想：锁、秘密、童年、父亲、知识、开启、关闭...
3. **符号库构建**：个人符号库（达利：蚂蚁=死亡，软表=时间，抽屉=隐藏）
4. **跨领域组合**：
   - 将艺术符号用于商业广告（软表形状 → 手表广告）
   - 将科学概念用于艺术（DNA双螺旋 → 《原子的丽达》）

**AI 增强版**：
```python
def subconscious_engine(dream_text):
    # Step 1: AI 提取意象（GPT-4 Vision 或 Claude）
    symbols = ai_extract_symbols(dream_text)  # ["钥匙", "门", "黑暗", "楼梯"]
    
    # Step 2: 自由联想（知识图谱）
    associations = []
    for symbol in symbols:
        associations += knowledge_graph.associate(symbol, depth=3)  # 每symbol 5-10个联想
    
    # Step 3: 符号组合+评分
    combinations = cross_combine(associations)
    scored = [(combo, creativity_score(combo)) for combo in combinations]
    top5 = sorted(scored, key=lambda x: x[1], reverse=True)[:5]
    
    return top5  # [(组合, 创意的), ...]
```

### 3. 跨界融合决策矩阵

**达利跨界成功案例**：
| 领域 | 项目 | 跨界得分 | 结果 |
|------|------|----------|------|
| 电影 | 《一条安达鲁狗》 | 0.78 | 电影史经典 |
| 广告 | Lanvin 香水广告 | 0.73 | 高佣金，品牌溢价 |
| 产品 | 达利沙发（扭曲设计） | 0.65 | 奢侈品家具 |
| 科学 | DNA 双螺旋艺术化 | 0.82 | 获得科学界认可 |
| 出版 | 《堂吉诃德》插图 | 0.75 | 经典再版 |

**跨界得分公式**：
```
跨界得分 = (受众重叠度 × 0.3) + (品牌契合度 × 0.3) + (收入潜力 × 0.2) + (技能迁移度 × 0.2)
```
- 受众重叠度：新领域受众与现有受众重叠（0-1）
- 品牌契合度：是否符合核心brand identity（0-1）
- 收入潜力：市场大小 + 定价能力（0-1）
- 技能迁移度：现有技能可迁移性（0-1）

**阈值**：
- >0.7：全力跨界，投入时间 30%+
- 0.5-0.7：试点项目，投入时间 10-20%
- <0.5：放弃或重新评估

**应用**：AI 艺术家考虑是否接商业广告
- audience_overlap: 0.8（AI 艺术爱好者 vs 广告观众重叠）
- brand_fit: 0.7（超现实 AI 符合品牌）
- revenue: 0.6（广告单笔 $50k）
- skill_transfer: 0.9（AI prompt engineering → 创意生成）
- **score = 0.74** → 接！

### 4. 自我神话构建（高管个人品牌）

**达利品牌 4 要素**：

1. **核心形象**：疯狂天才 + 精确技法（矛盾统一）
2. **视觉符号**：胡子（夸张向上翘）、手杖、高跟鞋、宽檐帽、豹纹披肩
3. **行为 stunts**：带疯狗出席发布会、用吻舔画布、装睡在画廊、与加拉 public 情欲
4. **争议叙事**：公开批评毕加索、"我和疯子唯一区别是我不疯"、政治立场频繁切换

**个人品牌构建模板**（非艺术领域）：
- **Step 1**：定义核心标签（3个词，如"数据诗人"、"AI 萨满"、"疯狂科学家"）
- **Step 2**：设计视觉符号（服装、配饰、发型、颜色）
- **Step 3**：策划 3-5 个争议话题（每季度1个）
- **Step 4**：媒体投放策略（每季度1次大事件）

**预算**（1年）：

-  stunt 策划：$20k
- 媒体：$50k（PR agency）
- 内容制作：$30k

**预期回报**：个人品牌溢价 + 客户信任度 + 媒体曝光价值 > $500k/年
