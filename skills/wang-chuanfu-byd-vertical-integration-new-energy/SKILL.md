## 核心框架

### 1. 垂直整合决策引擎

**目标**：判断哪些核心技术必须自制，哪些可以外购

**决策公式**：
```
vertical_integration_score = (tech_strategic×0.4) + (cost_sensitivity×0.3) + (supply_chain_risk×0.3)
```

**阈值**：
- 评分 > 0.7：强烈建议垂直整合
- 评分 0.4-0.7：部分整合，保留备份供应商
- 评分 < 0.4：外购优先

**关键指标**：
- 技术重要性（战略控制、差异化）
- 成本敏感度（占整机成本比例）
- 供应链风险（供应商数量、地缘政治）

**适用场景**：
- 电动车三电系统
- 手机 SoC + 射频 + 屏幕
- 机器人核心执行器
- 任何硬件产品的核心模块

### 2. 电池技术演进路线图

**核心选择**：磷酸铁锂（LFP） vs 三元锂（NCM/NCA）

| 维度 | 磷酸铁锂 | 三元锂 | BYD 选择 |
|------|---------|--------|----------|
| 成本 | 低 30% | 高 | ✅ LFP |
| 安全性 | 高（不燃） | 中（热失控风险） | ✅ LFP |
| 能量密度 | 中（160-180Wh/kg） | 高（240-280Wh/kg） | ⚠️ 通过结构创新弥补 |
| 循环寿命 | 长（3000+次） | 中（1500-2000次） | ✅ LFP |

**BYD 的逆向创新**：
- 在能量密度不占优的情况下，通过 **CTP（Cell to Pack）** 技术提升体积利用率
- 通过 **Blade Battery（刀片电池）** 结构创新，兼顾安全与能量密度
- 主打 **安全性 + 成本 + 寿命** 组合拳，而非单一参数竞赛

**路线图建议**：
1. 起步阶段：选择成熟、低成本技术（如 LFP）
2. 规模化后：通过结构创新提升性能，而非追逐最新化学体系
3. 高端市场：可考虑三元锂或固态电池作为补充

### 3. 全产业链自研系统

**BYD 的三电全部自制**：
- **电池**：自产电芯、Pack、BMS
- **电机**：永磁同步电机，自研定转子
- **电控**：VCU、MCU、BMS 三合一
- **半导体**：IGBT 芯片自制（2015 年突破）

**协同效应**：
- 成本降低：省去中间商利润，内部结算价格优化
- 快速迭代：三电协同调优，新车开发周期缩短 30-50%
- 供应安全：2020-2022 年芯片短缺时期，IGBT 自供减少受影响
- 技术保密：核心算法、材料配方不外泄

**实施路径**：
1. 识别核心技术模块（占成本 >20% 或战略重要性高）
2. 分阶段自研：先成熟采购 → 合作开发 → 完全自制
3. 建立内部供应商体系，模拟外部竞争机制
4. 开放部分非核心模块给外部供应商，保持生态健康

### 4. 政策借力与市场 Timing

**中国新能源政策周期**：
- 2009-2012：试点示范，补贴高但市场小
- 2013-2018：快速扩张，补贴退坡但双积分启动
- 2019-2022：补贴大幅减少，双积分驱动传统车企转型
- 2023+：补贴完全退出，市场化竞争

**BYD 的借力策略**：
- 2008 年推出首款电动车（F3DM），赶上了第一波补贴
- 2015 年后逐步降低补贴依赖，靠成本优势竞争
- 2021 年借助特斯拉涨价潮，抢占 10-20 万价格带
- 2023 年后海外扩张（东南亚、欧洲、拉美）

**Timing 判断**：
- 政策红利期：快速上量，占领市场份额
- 政策退坡期：提前 1-2 年准备好成本优势，确保盈利
- 市场化期：靠产品力 + 成本 + 品牌三要素竞争

### 5. 成本压缩与规模化

**BYD 的成本控制金字塔**：

```
原材料采购（40%成本压缩）
  ├── 锂、钴、镍等大宗商品长单
  ├── 自有锂矿（非洲、盐湖）
  └── 规模采购折扣（300万辆/年）

生产工艺（30%成本压缩）
  ├── 自动化生产线（每万人产能提升 50%）
  ├── 工艺简化（减少工序，提高良率）
  └── 能耗管理（光伏 + 储能自供）

研发摊销（20%成本压缩）
  ├── 平台化（e平台3.0覆盖多车型）
  ├── 零部件通用化（70% 通用率）
  └── 快速迭代摊薄研发费用

管理与物流（10%成本压缩）
  ├── 垂直整合减少供应链成本
  ├── 自建物流车队
  └── 数字化管理降低人工
```

**规模化拐点**：
- 电动车：年产能 50 万辆时，边际成本开始显著下降
- 电池：GWh 级产能利用率 >80% 时，成本曲线陡峭下降
- 整车：150 万辆/年时，采购议价权达到平台期
