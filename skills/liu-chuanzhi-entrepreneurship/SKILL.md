# Skill 57: 柳传志创业管理系统

**核心公式**:


## 核心模块

### 1. 拐大弯决策模型

**概念**: 在关键决策上避免急转弯，预留足够时间和空间进行验证和调整。

**算法**:
```
turn_radius_score = (decision_importance × impact_magnitude) / (time_pressure × uncertainty)
- turn_radius_score < 0.3: 急转弯（高风险）
- turn_radius_score 0.3-0.7: 中等转弯（需要监控）
- turn_radius_score > 0.7: 大弯（安全）
```

**Checklist**:
- [ ] 决策重要性评估（1-10分）
- [ ] 影响范围评估（1-10分）
- [ ] 时间压力评估（1-10分）
- [ ] 不确定性评分（1-10分）
- [ ] 计算拐大弯指数，决定是否需要延后或分阶段

**适用场景**: 产品方向调整、融资决策、市场进入、组织变革

### 2. 搭班子五要素

**核心**: 建团队不是招人，是"搭班子"。

**五道标准**:
1. **德才兼备**（权重30%）：品德优先，能力次之
2. **能力互补**（权重25%）：技能、经验、性格互补，避免同质化
3. **有决策能力**（权重20%）：能在信息不全时做出判断
4. **年轻有潜力**（权重15%）：年龄结构合理，有成长空间
5. **诚信透明**（权重10%）：言行一致，信息共享意愿

**班子健康度评分**:
```
team_health = (德×0.3 + 才×0.25 + 互补×0.2 + 决策×0.15 + 年轻×0.1) × 诚信系数(0.8-1.2)
```

**搭建流程**:
1. 明确核心岗位（CEO、CTO、COO、CFO、CMO）
2. 按五道标准评估候选人
3. 试用期3个月，观察协作和决策
4. 定期（季度）班子复盘

### 3. 定战略三原则

**原则**:
1. **切合实际**: 战略要基于自身资源和能力，不盲目对标
2. **分阶段实施**: 3年愿景、1年计划、季度目标层层分解
3. **动态调整**: 每季度复盘，必要时修正但避免频繁转向

**战略文档模板**:
```markdown
## 愿景 (3-5年)
- 我们要成为...

## 使命
- 我们为什么存在...

## 年度目标 (SMART)
1. 财务: 营收、利润、现金流
2. 产品: 版本号、用户数、NPS
3. 市场: 份额、地域、客户群

## 关键举措 (3-5个)
- 举措1: ...
- 举措2: ...

## 资源配置
- 人力: ...
- 资金: ...
- 技术: ...

## 风险与应对
- 风险1: ... 应对: ...
```

### 4. 带队伍方法论

**三层培训体系**:
- **入职培训**: 企业文化、核心价值观、基本技能
- **在岗实践**: 导师制（每新人配导师）、项目实战
- **领导力发展**: 梯队培养（高潜人才计划）

**激励机制**:
- 物质激励: 薪资、奖金、股权/期权（联想模式：员工持股会）
- 精神激励: 荣誉、认可、使命感
- 成长激励: 培训、轮岗、晋升通道

**文化塑造**:
- **说到做到**: 承诺必须兑现，建立信任
- **求实进取**: 实事求是，追求卓越
- **团队协作**: 共同目标，互相支持

### 5. 复盘文化

**复盘四步法**:
1. **目标回顾**: 当初的目标是什么？
2. **结果评估**: 实际达成如何？数据对比
3. **原因分析**: 成功/失败的根本原因（5 Why）
4. **经验提炼**: 可复制的经验？要改进的点？

**频率**:
- 项目结束后: 必须复盘（大复盘）
- 每季度: 战略复盘
- 每月: 运营复盘
- 每周: 团队站会（小复盘）

**复盘文档模板**:
```markdown
# [项目/事件]复盘报告

## 目标
- 原定目标: ...

## 结果
- 实际结果: ...
- 差异分析: ...

## 原因
- 成功因素: ...
- 失败因素: ...
- 根本原因: ...

## 学习
- 经验教训: ...
- 行动项: ...
```

## 实施 Playbook

### 创业公司（<100人）
1. **阶段1**: 搭核心班子（3-5人，按五道标准）
2. **阶段2**: 定第一年战略（聚焦产品-市场匹配）
3. **阶段3**: 建立复盘节奏（每周站会，每月复盘）
4. **阶段4**: 股权设计（参考联想员工持股会）

### 成长期（100-1000人）
1. 班子扩充，设部门负责人
2. 战略分3年规划，年度分解
3. 建立培训体系和导师制
4. 文化固化（价值观考核）

### 成熟期（>1000人）
1. 班子的决策能力和年轻化
2. 战略多元化，业务单元制
3. 领导力梯队建设
4. 复盘制度化，知识管理
