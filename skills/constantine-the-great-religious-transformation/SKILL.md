## 🎯 技能概述

**核心公式**: **帝国重生 = (军事胜利合法性 × 宗教政策工具) + (首都重构 × 行政改革) + (货币稳定 × 王朝规划)**

君士坦丁大帝（Constantine the Great，272-337 AD，统治306-337年）是罗马帝国晚期最关键转型者：

- **宗教转型**：312年米尔维安大桥 vision → 313年米兰敕令 → 325年尼西亚会议，将基督教从受迫害者变为国教
- **首都重构**：330年新建君士坦丁堡（拜占庭），战略位置+贸易+防御，东罗马存续1000年
- **行政改革**：完善四帝共治，行省细分，官僚专业化
- **货币改革**：引入 solidus 金币，纯度保持300年，恢复经济信心
- **王朝规划**：指定三子共治，但未解决多子争权（埋下分裂种子）

**适用场景**：
- 组织危机后合法性重建（利用外部价值观）
- 企业/国家品牌重塑（符号、叙事）
- 总部迁移或地理战略重构
- 行政体系改革与效率提升
- 货币或定价体系稳定化
- 家族企业多子女继承规划

**差异化价值**：
- 唯一整合“军事-宗教-首都-行政-货币-继承”六维转型
- 提供“宗教工具化”从宽容到国教的完整路径
- 首都迁移决策公式（capital_relocation_score）
- 金币稳定性原则（solidus 模型）
- 历史影响持续2000年（基督教+东罗马）

## 📐 核心算法与公式

### 1. 军事-宗教合法性转化器

**背景**：君士坦丁米尔维安大桥前夜见异象，借基督教上帝获得胜利合法性。

**输入**：
- `military_success_prob`：军事胜利概率（0-1）
- `religious_group_size`：宗教群体规模（0-1，占总人口）
- `ideological_fit`：意识形态与组织价值观契合度（0-1）
- `existing_legitimacy`：当前合法性基础（0-100）

**计算**：
```
legitimacy_score = (military_success_prob × 40) + (religious_group_size × 40) + (ideological_fit × 20)
# 结果 0-100
```

**应用**：
- legitimacy_score > 80 → 可借宗教大幅提升合法性
- legitimacy_score 50-80 → 可尝试局部合作
- legitimacy_score < 50 → 需先增强军事实力或寻找其他意识形态

**案例**：
- 君士坦丁：military_success_prob=0.7（以少胜多），religious_group_size=0.1（10%基督徒），ideological_fit=0.8 → legitimacy_score = 28+4+16=48 → 加上后续政治操作达标

---

### 2. 宗教政策工具化引擎

**三阶段模型**：
1. **Tolerance**（宽容）：米兰敕令（313）→ 所有宗教合法
2. **Favoritism**（优惠）：325年尼西亚会议 → 基督教优先，皇帝主导教义
3. **Control**（控制）：后期 → 基督教成为事实国教，其他宗教边缘化

**量化公式**：
```
religious_unity = (state_support × 0.5) + (doctrinal_uniformity × 0.3) + (institutional_control × 0.2)
```

**目标**：religious_unity > 0.8 实现高度统一（君士坦丁后期约0.75）

**策略**：
- State support: 免税 + 教会建筑 + 神职人员薪酬
- Doctrinal uniformity: 召开公会议（Nicene Creed）强制统一
- Institutional control: 皇帝介入主教任命（Caesaropapism）

---

### 3. 首都重构与地理政治引擎

**迁移决策公式**：
```
capital_relocation_score = (strategic_location × 0.3) + (economic_potential × 0.3) + (defensibility × 0.2) + (symbolic_value × 0.2)
```

**阈值**：>0.7 强烈建议迁移；0.5-0.7 可考虑；<0.5 维持原址

**君士坦丁堡得分（330年）**：
- strategic_location: 0.9（控制博斯普鲁斯，欧亚枢纽）
- economic_potential: 0.85（金角湾贸易）
- defensibility: 0.95（三面环水，陆墙易守）
- symbolic_value: 0.7（新罗马象征）
- **score = 0.81 → 迁移成功！**

**后续影响**：东罗马存续1000年（1453 fall）

---

### 4. 行政改革与官僚专业化系统

**三层结构**：
- Province (<10k pop) → Diocese (regional group) → Praetorian Prefecture (>1M pop)
- Civil service vs Military command 分离

**效率公式**：
```
administrative_efficiency = (bureaucrat_ratio × 0.4) + (separation_of_powers × 0.3) + (tax_collection_efficiency × 0.3)
```

**目标**：efficiency > 0.8，成本增长 < GDP growth

**君士坦丁实践**：
- bureaucrat_ratio: 0.7（适度）
- separation: 0.8（文官武官分离）
- tax_efficiency: 0.75
- **efficiency ≈ 0.76**（良好）

---

### 5. 货币稳定与信心恢复

**Solidus 原则**：
- 金币 4.5g，纯度保持 >99% for 300 years
- 对抗 inflation，恢复 trade confidence

**信心指数**：
```
confidence_index = (purity × 0.5) + (acceptance × 0.3) + (inflation_rate × -0.2)
```

**目标**：>0.85

**君士坦丁**：purity=0.99, acceptance=0.8, inflation≈低 → confidence ≈ 0.91

---

### 6. 王朝继承规划（多子制衡改进版）

**基于塞维鲁失败**：明确分割 + 第三方仲裁 + 长子 senior Augustus 但不是绝对

**继承规划评分**：
```
succession_plan_score = (early_announcement × 0.2) + (clear_territorial_division × 0.3) + (arbitration_mechanism × 0.3) + (senate_blessing × 0.2)
```

**君士坦丁**：提前宣布三子为 Caesar → Augustus，分治西、中、东；但无明确仲裁机制 → score ≈ 0.7（中等）

**结果**：死后帝国短暂统一后再次分裂（侄子 vs 儿子），说明继承规划仍需加强。

## 🛠️ 使用示例

### 示例：企业危机后合法性重建

**背景**：企业遭遇重大丑闻，合法性跌至30%，需重建信任。

**应用**：
1. **军事-宗教合法性转化**：
   - military_success_prob: 当前无，但可打造"改正行动"成功案例
   - religious_group_size: 0（企业无宗教）
   - ideological_fit: 0.9（价值观驱动）
   - legitimacy_score = 0 + 0 + 18 = 18 → 太低
   - 改进：先获得重大客户/合作伙伴支持（模拟军事胜利）， legitimacy 提升至 50+

2. **宗教政策工具化**：转化为"价值观共同体"
   - 定义核心价值观（类似教义）
   - 全员培训（类似洗礼）
   - 高层成为价值观化身（类似皇帝为教会之父）

3. **总部重构**：若原总部受损，迁移至新城市（象征新开始）

4. **行政改革**：透明化 + 第三方审计

5. **货币稳定**：承诺薪酬+福利稳定（如金币）

6. **继承规划**：明确第二代接班人

**预期**：2-3年恢复合法性至70%+

---

### 示例2：国家迁都决策

**背景**：某国考虑迁都，需评估可行性。

**应用 capital_relocation_score**：
- strategic: 0.8（内陆中心）
- economic: 0.7（发展潜力大）
- defensibility: 0.9（易守）
- symbolic: 0.6（象征意义）
- **score = 0.76 → 支持迁移**

同时评估：
- 宗教统一：religious_unity > 0.8（避免宗教冲突）
- 行政效率：保证效率不降
- 货币稳定：迁都期间保持币值稳定
