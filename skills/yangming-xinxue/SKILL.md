# 王阳明心学决策引擎

## 📋 概述

**王阳明心学决策引擎**是一个将明代哲学家王阳明（1472-1529）的心学思想转化为现代决策工具的系统。它的核心是"知行合一""致良知"两大基石，帮助领导者在复杂环境中做出既理性又符合道德本心的决策。

> "知之真切笃实处即是行，行之明觉精察处即是知。" —— 王阳明

## 🎯 核心功能

### 1. 知行一致性检查
- **动机-行动对齐**：决策前审视"为何做"与"怎么做"是否统一
- **真知假知鉴别**：区分"了解知识"与"践行知识"
- **延迟检验**：设置3-6个月回顾点，验证知行是否脱节

### 2. 良知扫描仪 (Liangzhi Scanner)
- **内在道德 GPS**：识别内心真正的道德判断，而非社会惯性
- **私欲过滤器**：检测情绪、偏见、利益对决策的扭曲程度
- **心安测试**：决策后是否"此心光明"，还是辗转难眠

### 3. 事上磨练模拟 (Shi Shang Mo Lian)
- **情境压力测试**：模拟真实困难环境，练习"在事上练心"
- **逆境决策演练**：危机、资源不足、高压下的良知指引
- **日常事务精进**：将琐事视为修炼场，避免"静坐空谈"

### 4. 战略致良知 (Strategic Liangzhi)
- **大决策三问**：
  1. 这是否符合天理良知？（道德维度）
  2. 我是否有真知灼见？（认知维度）
  3. 我能否行道到底？（执行维度）
- **团队良知校准**：组织使命与成员良知是否共振
- **长期主义检验**：短期利益是否腐蚀长期良知

## 🧮 核心公式

```javascript
// 良知综合得分
liangzhi_score = (moral_fit * 0.4) + (practical_feasibility * 0.3) + (long_term_impact * 0.3)

// 知行一致性
action_alignment = intention_purity * knowledge_depth * execution_capability

// 私欲系数（越高越危险）
selfishness_level = (personal_gain - social_value) / (personal_gain + 0.001)

// 最终决策
if (liangzhi_score >= 0.8 && action_alignment >= 0.7 && selfishness_level <= 0.3) {
  决策 = "执行";
} else if (liangzhi_score >= 0.6) {
  决策 = "暂停·反思·再评估";
} else {
  决策 = "放弃·重新校准良知";
}
```

**参数解释：**
- `moral_fit`：决策是否符合天理良知（0-1，需良知扫描仪打分）
- `practical_feasibility`：现实可行性（资源、团队、时机）
- `long_term_impact`：长期正向影响（3-5年视野）
- `intention_purity`：动机纯度（是否夹杂私欲）
- `knowledge_depth`：是否"真知"（看书≠真知，实践出真知）
- `execution_capability`：执行能力（资源、能力、团队）
- `personal_gain`：个人/公司直接利益
- `social_value`：对社会、客户、员工的贡献价值

## 📊 使用示例

### 示例 1：创业公司是否接受巨额融资

```yaml
场景：AI创业公司，估值$50M，投资人出价$100M（double）
融资用途：扩张团队，抢占市场
潜在问题：失去控制权，价值观冲突

良知扫描：
- moral_fit：融资是否"致良知"？投资人价值观是否匹配？（得分0.6）
- practical_feasibility：资金到位快，团队可扩张（得分0.8）
- long_term_impact：快速成功但可能迷失使命（得分0.5）

liangzhi_score = (0.6×0.4)+(0.8×0.3)+(0.5×0.3)=0.24+0.24+0.15=0.63

知行一致性：
- intention_purity：初心是改变世界，不是套现（0.9）
- knowledge_depth：理解融资条款的影响（0.7）
- execution_capability：团队能驾驭快速扩张（0.8）
- action_alignment = 0.9×0.7×0.8=0.504

私欲系数：
- personal_gain：创始团队财务自由（+10）
- social_value：产品真正服务用户（+8）
- selfishness_level = (10-8)/(10) = 0.2

决策：⏸️ 暂停·再评估
理由：良知得分0.63（中等），知行一致性0.504偏低（需加强真知，深入理解条款），需与投资人深度沟通价值观对齐。
```

### 示例 2：高管是否揭露公司问题（风险情景）

```yaml
场景：发现产品存在安全隐患，但曝光会引发股价暴跌、公司动荡

良知扫描（某人内心）：
- moral_fit：隐瞒=害人，揭露=良知（得分0.9）
- practical_feasibility：曝光后可能被解雇、行业封杀（得分0.3）
- long_term_impact：短期灾难，长期修复信任（得分0.8）

liangzhi_score = 0.36+0.09+0.24=0.69

知行一致性：
- intention_purity：保护用户安全（0.95）
- knowledge_depth：了解法律、技术漏洞影响（0.8）
- execution_capability：有证据、有计划应对（0.7）
- action_alignment = 0.95×0.8×0.7=0.532

私欲系数：
- personal_gain：保住工作、声誉（-5，实际是损失）
- social_value：用户安全、行业透明（+10）
- selfishness_level = (-5-10)/(-5) = 3.0（负值代表牺牲自己，绝对值大说明高度无私）

决策：✅ 执行（揭露）
理由：良知得分0.69，知行一致性中等但动机纯正，私欲系数为负（自我牺牲）。王阳明会说："知行本体，只在致良知。"此时良知昭然，虽千万人吾往矣。
```

### 示例 3：团队是否加班赶工项目

```yaml
场景：客户 deadline 逼近，需全员加班1个月

良知扫描（管理者角度）：
- moral_fit：尊重员工福祉 vs 承诺客户（得分0.5）
- practical_feasibility：时间紧，只能靠加班（得分0.7）
- long_term_impact：团队 burnout，人才流失（得分0.3）

liangzhi_score = 0.2+0.21+0.09=0.50

知行一致性：
- intention_purity：想完成任务，但也关心员工（0.6）
- knowledge_depth：了解加班的代价（0.8）
- execution_capability：有备用方案（外包、分阶段）（0.6）
- action_alignment = 0.6×0.8×0.6=0.288

私欲系数：
- personal_gain：保住客户、个人业绩（+6）
- social_value：员工健康、公司文化（+4）
- selfishness_level = (6-4)/(6)=0.333（略高于阈值0.3）

决策：⏸️ 暂停·再评估
理由：良知得分0.5偏低，知行一致性差（知健康重要但行不通），私欲系数超标。需要重新"致良知"：是否可以外包？是否可以分阶段交付？是否可以与客户重新协商？
```

## 📈 评估框架

### Step 1：良知扫描仪 (Liangzhi Scanner)

**使用5个核心问题：**

1. **天理问**：这个决策符合天理良知吗？是否符合"是非、善恶、美丑"的普遍标准？
2. **动机问**：我的初心是什么？是否夹杂私欲（面子、金钱、权力）？
3. **感受问**：内心是"此心光明"还是"辗转难眠"？
4. **后果问**：对客户、员工、社会、环境的影响总体是正向的吗？
5. **长期问**：5年后回顾，我会为此骄傲还是后悔？

**评分方法（每个问题 0-1，取平均）：**
- 0.9-1.0：良知昭然，如皓月当空
- 0.7-0.9：良知基本清晰，有小干扰
- 0.5-0.7：良知被遮蔽，需进一步"格物致知"
- 0.0-0.5：良知不明，暂停决策

### Step 2：知行一致性检验

**检查三个层面：**

1. **知是否真知？**
   - 是书本知识还是亲身实践得来的？
   - 是否理解底层逻辑而非表面现象？
   - 是否有成功/失败案例验证？

2. **行是否匹配？**
   - 当前行动是否直接服务于"知"？
   - 是否存在"说一套做一套"？
   - 是否为了短期利益偏离长期认知？

3. **知与行是否循环强化？**
   - 行动后是否反思、总结、提升认知？
   - 是否在"事上磨练"中深化理解？
   - 形成"知→行→思→知"的飞轮？

**评分：** 三者乘积，每个 0-1

### Step 3：私欲过滤

**识别私欲类型：**
- **名利欲**：追求个人声望、财富、地位
- **情绪欲**：愤怒、恐惧、贪婪、嫉妒主导
- **舒适欲**：回避困难、选择轻松但无价值的事
- **从众欲**：随大流，不敢标新立异

**计算方法：**
```
私欲系数 = (个人直接收益 - 社会总价值) / (个人直接收益 + 0.001)
```
- 结果 < 0：自我牺牲，无私决策
- 结果 0-0.3：可接受范围
- 结果 0.3-0.5：私欲较重，需警惕
- 结果 > 0.5：被私欲主导，决策危险

### Step 4：综合决策

使用核心公式计算，对照阈值判断。

### Step 5：事上磨练计划

如果"暂停·再评估"，制定：
- 良知校准行动（阅读经典、静思、与导师交流）
- 知行合一练习（小规模试验、快速反馈）
- 私欲克治方法（记录情绪、设立监督人）

## 🎓 应用场景

### 1. 领导力决策
- 高管晋升评估：是否"致良知"且知行合一
- 战略取舍：进入新市场是否符合公司使命
- 危机公关：真相与面子、短期与长期的良知抉择

### 2. 团队管理
- 绩效评估：是否公正无私（良知），评价是否与行为一致（知行）
- 文化建设：团队是否"人人致良知"，还是内卷政治
- 冲突调解：双方谁更符合良知，谁在推诿

### 3. 产品/商业决策
- 是否推出有缺陷但能赚钱的功能？
- 广告是否夸大误导？
- 客户服务是否真正为客户创造价值？

### 4. 个人成长
- 职业选择：高薪但违背内心 vs 低薪但符合良知
- 学习选择：学什么能"知行合一"（实践导向）
- 生活习惯：戒掉"假努力"，真实践

### 5. 投资决策
- 是否投资有道德争议的行业（烟草、赌博）？
- 是否为了高回报忽视风险（贪婪）？
- 是否因恐惧错过机会？

## ⚠️ 常见错误

1. **将"良知"主观化**：每个人良知不同，没有标准 → 混淆道德相对主义
   - 解决：良知有普世基础（仁、义、礼、智、信），参考儒家共识

2. **静坐空谈，不行也不练**：变成口头哲学，不应用到实际
   - 解决：强制"事上磨练"环节，必须有实践行动

3. **急功近利**：期望一次"致良知"就彻底解决所有问题
   - 解决：接受渐进过程，"破山中贼易，破心中贼难"

4. **道德绑架**：用良知要求他人，自己不践行
   - 解决："躬自厚而薄责于人"，管理者先修己

5. **忽视知行差距**：低估"知易行难"的难度，计划过于理想
   - 解决：在实践中反复检验，持续调整
