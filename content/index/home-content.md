## 最新动态配置

首页的"最新动态"区域会自动从 `latest_news.js` 读取数据。

### 如何更新最新动态

#### 1. 更新成长日记
编辑 `latest_news.js` 中的 `growth` 对象：

```javascript
growth: {
    date: '2026-03-14',      // 日期
    title: '新日记标题',      // 标题
    summary: '日记摘要...',   // 摘要（100字以内）
    quote: '金句引用',        // 精彩句子
    emoji: '🚀'              // 表情图标
}
```

#### 2. 更新小说章节
编辑 `latest_news.js` 中的 `novel` 对象：

```javascript
novel: {
    chapter: 11,             // 章节号
    title: '章节标题',        // 标题
    date: '2027-09-01',      // 日期
    excerpt: '章节摘要...',   // 摘要
    quote: '精彩句子'         // 引用
}
```

#### 3. 添加社区动态
编辑 `latest_news.js` 中的 `community.updates` 数组：

```javascript
updates: [
    { text: '动态内容', time: '2小时前', type: 'project' },
    // type 可选：project/novel/member/tutorial/other
]
```

### 数据来源

- **成长日记**：从 `content/growth/` 文件夹读取最新一篇
- **AI小说**：从 `novels_data.js` 读取最新章节
- **社区动态**：从 localStorage 读取用户活动 + 手动配置

### 自动更新机制

当用户在社区进行以下操作时，会自动更新动态：
- 发布新项目 → 添加 project 类型动态
- 发表评论 → 更新评论统计
- 登录 → 添加 member 类型动态（首次）
