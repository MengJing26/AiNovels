## ▣ 部署步骤（Cloudflare Pages）

1. 上传代码到 GitHub
2. 注册 Cloudflare 账号
3. Pages → 创建项目 → 连接 GitHub
4. 选择仓库，构建设置留空
5. 部署完成，获得 xxx.pages.dev 域名
6. （可选）绑定自定义域名
